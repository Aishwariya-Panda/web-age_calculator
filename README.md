# Age Calculator Web Application #

link: https://web-age-calculator-1.onrender.com/

Project Overview:
This is a simple and responsive Age Calculator web application developed using HTML, CSS, and JavaScript. The application calculates a user's age in Years, Months, and Days based on the Date of Birth entered in DD-MM-YYYY format.

  Features:
- User-friendly interface
- Manual date input in DD-MM-YYYY format
- Accurate age calculation (Years, Months, Days)
- Input validation
- Error handling for incorrect or future dates
- Clean and responsive design

 Languages Used:
- HTML (Structure)
- CSS (Styling)
- JavaScript (Logic & Age Calculation)

  How It Works:
1. User enters Date of Birth in DD-MM-YYYY format.
2. JavaScript splits the input and converts it into a Date object.
3. Current system date is fetched.
4. The difference is calculated and displayed in years, months, and days.
